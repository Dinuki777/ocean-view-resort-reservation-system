# 🌊 Ocean View Resort Reservation System

## 🎯 Project Overview

The **Ocean View Resort Reservation System** is a distributed **web-based hotel management application** designed to manage room reservations efficiently.
The system enables staff members to create, view, update reservations, and calculate guest bills while maintaining secure session management.

This project demonstrates the implementation of **web services, database integration, software design patterns, and version control practices**.

---

✅ User Authentication System

✅ Add and manage reservations

✅ Generate reservation and revenue reports

# 🏗️ System Architecture

The system follows a **3-Tier Architecture** to ensure scalability and maintainability.

### 1️⃣ Presentation Layer

Handles the user interface.

Technologies used:

* HTML5
* CSS3
* JavaScript

---

### 2️⃣ Application Layer

Processes business logic and handles communication between the frontend and database.

Technologies used:

* Java Servlets
* RESTful Web Services

---

### 3️⃣ Data Layer

Stores and manages system data.

Technology used:

* MySQL 8.0 Database

Database includes tables for:

* Users
* Reservations
* Rooms
* Billing

---

# 🧩 Design Patterns Used

### MVC (Model View Controller)

Separates the application into:

* **Model** – Data and business logic
* **View** – User interface
* **Controller** – Handles user requests

### DAO (Data Access Object)

Handles all database operations and queries.

### Singleton Pattern

Ensures only **one instance** of service classes exists.

### Factory Pattern

Creates service objects dynamically without exposing object creation logic.

# ⚙️ Installation Steps

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Dinuki777/ocean-view-resort-system.git
```

### 2️⃣ Import Project

Open **Eclipse IDE** and import the project as a **Dynamic Web Project**.

### 3️⃣ Create Database

Create a MySQL database:

```
ocean_view_resort_db
```

Then import the provided SQL file.

### 4️⃣ Configure Apache Tomcat

Add **Apache Tomcat Server** in Eclipse and deploy the project.

### 5️⃣ Run the Application

Open a browser and run:

```
http://localhost:8080/OceanViewResort
```
