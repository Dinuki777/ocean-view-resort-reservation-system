Help Section - Staff Guidelines

Welcome to the Ocean View Resort Reservation System! This guide will help new staff members navigate and use the system effectively.

1. User Authentication (Login)

  1. Enter your username and password on the login page.
  2. Click the "Login" button to access the system.
  3. If you forget your password, contact the system administrator.
  4. Always log out when leaving the workstation.

2. Add New Reservation

  1. Navigate to "Add Reservation" from the menu.
  2. Fill in all required fields:
    • Room Number (e.g., 101, 102, 201)
    • Guest Name (full name as per ID)
    • Address (complete residential address)
    • Contact Number (mobile number)
    • Room Type (Single/Double/Deluxe)
    • Check-In Date (arrival date)
    • Check-Out Date (departure date)
  3. Click "Add Reservation" to save the booking.
  4. The system will auto-generate a unique Booking Number.

3. Display Reservation Details

  1. Go to "Display Reservation" page.
  2. All reservations are displayed in a table format.
  3. To search for a specific reservation:
    • Enter the Booking Number in the search box.
    • Click "Find" to retrieve the record.
  4. To delete a reservation:
    • Find the reservation using the Booking Number.
    • Click "Delete" and confirm the action.

4. Calculate and Print Bill

  1. Navigate to "Calculate Bill" page.
  2. Enter the Booking Number and Room Number.
  3. Click "Search" to view reservation details.
  4. Click "Calculate Bill" to compute the total cost.
  5. Room rates:
    • Single Room: Rs. 5,000 per night
    • Double Room: Rs. 8,000 per night
    • Deluxe Room: Rs. 12,000 per night
  6. The system calculates: (Number of Nights) × (Room Rate) = Total Price

5. Update Reservation

  1. Go to "Add Reservation" page.
  2. Enter the Booking Number in the search box.
  3. Click "Find" to load the existing reservation.
  4. Modify the required fields.
  5. Click "Update" to save changes.
Important Notes:
  • Always verify guest identification before accepting reservations.
  • Ensure Check-Out date is after Check-In date.
  • Double-check room availability before confirming bookings.
  • Maintain guest confidentiality at all times.
  • Report any system errors to the IT department immediately.
